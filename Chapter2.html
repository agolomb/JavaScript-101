Naming variables

Generally we can name variables anything we want but there's important rules we must follow.

  • They can't start with integers.
  • The only special characters they can have are $ or _.
  • You can't name a variable "let", "const", or "var". These are KEYWORDS in JavaScript that have special abilities.
  • There's lots of other keywords in JavaScript with special abilities. WE CANNOT use any of these to name our variables.
  • Examples: return, for, typeof, while,
  • There's many more, if the code breaks on your variable, check the name!

Control Flow

In JavaScript, the code is read from Top to Bottom.

Conditonal Execution

Sometimes we might not want our code to be read in a straight line top to bottom. We might want a certain thing to happen or something else to happen. We call this Conditonal Execution! We is if statements to take advantage of this. Look at the JavaScript to see this in action.

Type "squareMyNumber()" in the console to how this Conditional Execution works.

With conditional execution, we can check for more than one condition. We can use if, else if, and else to check lots of different conditions to decide what code we want to execute.
